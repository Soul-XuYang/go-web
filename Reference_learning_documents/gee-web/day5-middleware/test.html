Gee · Day05 测试面板

先在终端启动你的服务：go run . （监听 :9999）

1) 根路由：GET /

预期：返回 <h1>Hello Gee</h1>；控制台日志由全局 Logger 打印。

打开 /

2) v2 组（带 onlyForV2 中间件）：GET /v2/hello/:name

预期：中间件直接 Fail(500)，返回 {"message":"Internal Server Error"}；日志会有 “for group v2”。

3) error 组（AbortWith JSON）：GET /error/hello

预期：返回 500 与 {"message":"Error Path"}（由 RejectWith(500, ...) 中间件短路）。

打开 /error/hello
